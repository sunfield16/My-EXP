## 画像ブレンド
CGでは複数の画像が重なった場合に、様々な計算を行って  
最終的に描画される色を決める必要がある。  
画像ブレンドはこの計算の方法を指す。

様々な方法があり、場合に応じた使い分けが必要になる。

## 用語
* src ... 画像が重なった時、前に表示されている画像(前景)
* dst ... 画像が重なった時、後ろに表示されている画像(背景)
* RGB ... 各ピクセルRed, Green, Blueそれぞれの数値
* アルファ値(Alpha, A) ... 各ピクセルの透明度を表す(0.0 ~ 1.0)
    - 低いほど透明になる
