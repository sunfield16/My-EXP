「管理下のファイルはあるが`.git`だけ消滅した」という状況が発生した時に  
行った対応。  
ディレクトリによっては差分があり、単純なcloneでは問題になってしまう。

パッと出てきた方法として、以下2種類の方法を行った。

* 別の場所に同じリポジトリをcloneして、`.git`だけを[[cp]]などで輸送する
    - 差分が少なめだと予想できるならこっちの方が早いと思われる
* そのディレクトリにあるファイルを一旦退避して、cloneし直してから上書きする
    - 差分が大きすぎてresetとかにも一苦労しそうならこれが手軽
