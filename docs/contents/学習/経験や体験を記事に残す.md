この記事は言語化についての話がメインだが、  
その中に「経験」「体験」を残すことの効果についても書かれている。
```
**しかし、「経験」「体験」「成功」「失敗」などは誰かが記事に書かない限り増えることはありません。**

誰かの失敗を学び、失敗を回避することは成功方法を学ぶよりも
確実に成功へと近づきます。
誰かのやったうまくいった勉強法を学ぶことは、非効率な学習をして
時間を失うことを防ぎます。
```

「知識」は勉強したら身につくけど、「経験」や「体験」は  
知識を使って何かしないと身につかない。

この「経験」「体験」を記事やメモとして残すことは、誰かのためでもあるし未来の自分のためにもなると思われる。  
（やった事や学んだことを未来の自分が忘れてしまうことは多々あるため）

リードの知見やゲームしていて思ったことをメモすることはたまにやるが、  
これも「経験」「体験」となるんじゃないだろうか。
