補間用の係数により、2つの異なる数値の間を出力する方法。  
英語では`linear interpolation`と表される。

数値`A`と`B`、係数を`x`とすると以下のような式になる。  
（xの範囲は`0 <= x <= 1`）
```
y = A * x + B * (1 - x)
```

## 例
```
A = 10, B = 50 とする

x = 1 の場合
y = 10 * 1.0 + 50 * (1 - 1)
  = 10 + 50 * 0
  = 10

x = 0 の場合
y = 10 * 0 + 50 * (1 - 0)
  = 0 + 50 * 1
  = 50

x = 0.5 の場合
y = 10 * 0.5 + 50 * (1 - 0.5)
  = 5 + 50 * 0.5
  = 30
```

* [[アルファブレンド]]
* ゲームにおける[[座標移動]]
