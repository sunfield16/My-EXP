システム上でセキュリティ関連の監査を行う仕組み。  
デーモンを起動してリアルタイムに監査を行うほか、コマンドによりレポートの出力などが可能。

## レポートで見られる内容
* ユーザーのログインやログアウトなど操作
* ユーザーのコマンド実行
* ファイルへのアクセス
* システムイベント
