* Error: 実装ミスなどコード上で修正すべきものによるエラー
    - 使う側が気を付けるもの
    - Catchすることは想定しない
* Exception: 本来発生しないはずのエラー（つまり例外）
    - 関数を作った側が、使う側に状況を伝えるもの
    - Catchした方がいい
