英語では`Prepared Statement`と表記される、SQLに含まれる機能の1つ。  
SQLの構文を実行前に解析しておき、**クエリの構造が確定してから**パラメータを入れ込む。

例えば、MySQLでは以下のような感じで記述する。
```sql
-- 「stmt1」という名前のプリペアドステートメントを作成
-- 
PREPARE stmt1 FROM 'select * from cards where power >= ? and defence >= ?';

-- 後からパラメータを準備
SET @a = 5;
SET @b = 3;

-- パラメータを仕込んで実行
EXECUTE stmt1 USING @a, @b;
```

この仕組み上、[[SQLインジェクション]]を回避できるなど安全性の面でかなり強力。
