<https://oauth.net/>

Web・モバイル・デスクトップなどのアプリケーションから、  
シンプルかつ安全な認証を行うためのプロトコル。  
[[IETF]]によって策定された。

現在では[[OAuth2.0]]が使われており、今も開発が進められている。  
また、2.0の基本機能を統合・簡素化するための取り組みとしてOAuth2.1が進行中。

## ライブラリ
<https://oauth.net/code/>

OAuthを利用するためのOSSライブラリが各言語で開発されており、  
公式サイト上でもまとめられている。
