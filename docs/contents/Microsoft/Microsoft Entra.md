Microsoftが提供している[[IDaaS]]。

主にMicrosoft365や[[Azure]]などMicrosoftのサービスで使われるほか、  
[[Bing Ads API]]などの認証にも活用される。

## テナント
Entraにおける組織を表す。  
テナントには以下のようなことができる。  

* 内部・外部のユーザーを招待
* ユーザーを新規作成
* アプリを追加
