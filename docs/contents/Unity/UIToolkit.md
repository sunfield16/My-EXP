UGUIの後に出てきた、Unityの新しいUI作成フレームワーク。  
RectTransformなどは使わず、既存の作成方法から大きく変わる。

* どちらかというとWeb（HTMLレイアウト）っぽい組み方ができる
	- Position（絶対/相対座標）
	- Flexbox
	- Spacing（Margin, Padding）
* UGUIやIMGUIとの同時使用は可能
	- <https://docs.unity3d.com/Manual/UI-system-compare.html>
