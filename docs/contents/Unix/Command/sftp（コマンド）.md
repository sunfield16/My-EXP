[[SFTP（プロトコル）|SFTPプロトコル]]を使用して  
サーバー間のファイル転送や管理を行うコマンド。

[[scp]]がファイル転送のみを扱うのに対し、  
sftpはディレクトリリストの表示・ファイル削除などの操作も行える。
