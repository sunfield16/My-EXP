`tape archive`の略で、複数のファイルやディレクトリをまとめたアーカイブファイルを作成するコマンド。  
アーカイブ自体には圧縮などはかかっていないが、[[gzip]]などで圧縮が可能。

[[zip]]がUnix標準コマンドではないこともあり、バックアップを取る際に  
tar + gzipで圧縮して退避する方法が取られることは多い。

## 基本の使い方
```bash
# hogeディレクトリをtarでアーカイブ→gzで圧縮する
tar -czvf hoge.tar.gz hoge/
```

* オプションは全部繋げられる（`-`で1つずつ指定は不要）
    - `-f`は引数を取るので、繋げる場合は最後尾じゃないとダメ

## オプション
* -c: アーカイブを作成する（create）
* -x: アーカイブを展開する（extract）
* -z: gzipで圧縮・展開する
    - 展開する場合もこれを指定する必要あり
* -f: 作成するアーカイブのファイル名を指定する
* -v: 処理中のファイルを表示する（verbose）
