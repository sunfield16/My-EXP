<https://developer.mozilla.org/ja/docs/Web/CSS/object-fit>

CSSのプロパティの1つ。  
imgなどの画像がコンテナ内でどのように当てこまれるかを指定する。

例えば以下のような指定が可能。
* コンテナからはみ出たら収まるように縮小する
* コンテナからはみ出た部分をクリッピングする
* コンテナいっぱいに拡大表示する
