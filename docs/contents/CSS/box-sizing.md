<https://developer.mozilla.org/ja/docs/Web/CSS/Reference/Properties/box-sizing>

要素のサイズを算出するとき、どこまでを領域とするか指定する。  
widthやheightを100%で指定するときに、「padding含めて100%とするか」といった計算に影響する。

* content-box: 要素のサイズだけを含める
    - paddingやborderは別枠で計算
* border-box: 要素のサイズ・padding・borderを含める
