CSS上でサイズや長さを指定するのに使われる型。  
さまざまな単位で指定を行うことができる。

## 固定長
px以外はCSSで使われることはあまりない。

* cm ... 96px/2.54
* mm ... 1/10 cm
* in ... 2.54cm = 96px
* px ... 1/96 in

## 相対長
それぞれ他の要素を参照して、そこから数値が決まる。  

### フォントサイズに依存
ユーザー設定でフォントサイズが変わる場合がある。  
使い所はフォントサイズに合わせて要素も拡大したい場合など？

* em ... 親要素のフォントサイズ
    - 親要素のフォントサイズが16pxのとき、0.5emは8px
* rem ... ルート要素（`html`）のフォントサイズ

### ビューポートに依存
* vw ... ビューポートの幅（100vwで横幅ぴったり）
* vh ... ビューポートの高さ（100vwで縦幅ぴったり）

### 文字幅に依存
特定の文字の幅を基準として長さを決める。  
厳密に「横15文字分」のような表示枠を作りたい場合に使われる？

* ch ... その要素のフォントで、`0`の文字幅と同じ長さになる
* ic ... その要素のフォントで、`水`の文字幅と同じ長さになる

## パーセント表記
`%`での割合表記も可能。この場合は親要素の数値を参照する。  
（親要素の`witdh`が200pxのとき、50%を指定すると100px）
